#####################
## 화면 구성 / 기능 ##
#####################

< clientPage >
header
    제목, 로그인, 로그아웃, 회원가입, 비번찾기
sidebar
    로고, client 정보(비로그인시 IP정보, 클릭시 로그인 or 회원정보), 공지, 검색, 카테고리, 최근에 본 페이지
contents
    메인페이지 - 최신 글 목록, 최신 글 보기
    카테고리 선택페이지 - 카테고리 목록
    카테고리별 페이지 - 해당 카테고리 내 최신 글 목록, 최신 글 보기
    글 보기 페이지 - 글, 해당 글 앞뒤의 글 목록

< userPage >
joinPage
    이메일, 비밀번호, 비밀번호 확인, 이메일 인증(pass 가능)
loginPage
    이메일, 비밀번호, 비밀번호 찾기(인증시)
findPasswordPage
    이메일, 이메일 인증 완료 후 비밀번호 재 설정
userPage
    이메일 변경, 비밀번호 변경, 이메일 인증, 회원 탈퇴

< adminPage >
adminPage
    view통계 - 일별, 주별, 월별, 연별 view 숫자 보기
    블랙리스트 관리 - 조회/추가/삭제
    서버 cpu ram storage 확인 및 정리, 재시작
